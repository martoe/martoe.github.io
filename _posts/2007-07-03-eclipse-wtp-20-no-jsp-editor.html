--- layout: post title: 'Eclipse WTP 2.0: No JSP Editor' date: '2007-07-03T10:34:00.000+02:00' author: martin tags: - eclipse modified_time: '2012-04-29T10:51:07.110+02:00' blogger_id: tag:blogger.com,1999:blog-987786588703208935.post-1715870384454168017 blogger_orig_url: http://bxm-dev.blogspot.com/2007/07/eclipse-wtp-20-no-jsp-editor.html --- When switching to WTP 2.0 (based on Eclipse 3.3), make sure to use at least a Java 5 Runtime Environment. Otherwise there will be no XML editor, JSP editor, Javascript editor etc. available.
The Eclipse 3.3 platform “recommends” Java 5, but I didn’t find a statement that WTP requires Java 5. It took me 3 hrs to find it out
