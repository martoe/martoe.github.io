--- layout: post title: Using Vagrant behind a corporate proxy date: '2014-06-30T08:59:00.000+02:00' author: martin tags: modified_time: '2014-06-30T08:59:01.463+02:00' blogger_id: tag:blogger.com,1999:blog-987786588703208935.post-7380191445258319345 blogger_orig_url: http://bxm-dev.blogspot.com/2014/06/using-vagrant-behind-corporate-proxy.html --- 1. install vagrant-proxyconf plugin
set http_proxy=http://username:password@hostname:port
vagrant plugin install vagrant-proxyconf

2. configure the vagrant-proxyconf plugin
Create or edit file $HOME/.vagrant.d/Vagrantfile

Vagrant.configure("2") do |config|
  if Vagrant.has_plugin?("vagrant-proxyconf")
    config.proxy.http     = "http://username:password@hostname:port/"
    config.proxy.https    = "http://username:password@hostname:port/"
    config.proxy.no_proxy = "localhost,127.0.0.1,.example.com"
  end
end

3. download box and start vagrant
set HTTPS_PROXY=http://username:password@hostname:port
vagrant up
