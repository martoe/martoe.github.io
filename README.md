## Getting started on Windows

1. Download Jekyll:

        git clone git@github.com:madhur/PortableJekyll.git
        git clone git@github.com:martoe/martoe.github.io.git

1. Start Jekyll:

        cd PortableJekyll
        setpath.cmd
        cd ..\martoe.github.io
        jekyll serve --drafts

1. Open [localhost:4000](http://localhost:4000/)
